# pun-ci app

## Development

### Prerequisites

* Node.js >=16.4
* GNU Make
* Yarn

### Setup

```sh
; git clone git@github.com:pun-ci/app.git punci-app
; cd punci-app
; make init_dev
# Fill missing secret values in .env
; make dev
```

Open http://localhost:5000

### DB admin:

http://localhost:2113/
